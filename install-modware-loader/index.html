Installing Modware Loader

Oct 29th, 2012

Modware-Loader distribution provides command line scripts to import and export data from chado relational database in various formats.

  • Clone the Modware-Loader repository and switch to develop branch
  • Install the dependencies
  • Run any of modware command to list the sub-commands.
  • Run any of the subcommands to display its options

More or less any subcommands containing the word dicty is specific to work with D.discoideum chado oracle database.
